About the Game

Wooden Wars is a fast-paced realtime strategy game created for the University of Delaware's Game Development Club's Game Jam. The theme was "Toybox", so in this game you summon little wooden toy soldiers. I worked with my friend YakuzaBalooza over about two weeks for this game. We ended up winning the game jam with this submission!
